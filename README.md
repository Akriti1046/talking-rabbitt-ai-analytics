# Talking Rabbitt – Conversational AI Analytics

This repository contains the prototype idea for Talking Rabbitt, a conversational AI analytics platform.

Problem:
Business users struggle with complex dashboards and SQL queries when analyzing data.

Solution:
Talking Rabbitt allows users to ask natural language questions about their data and instantly receive answers and visual insights.

Example Question:
"Which region generated the highest sales?"

Features:
- Upload CSV dataset
- Ask questions about the data
- Automatic visualization

This project demonstrates the concept of "talking to data instead of building dashboards."
